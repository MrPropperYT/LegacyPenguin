# LegacyPenguin
a PC Launcher for Minecraft: Legacy Console Edition, which simplifies managing your MCLCE instances.

## Features
LegacyPenguin was built with being feature-rich in mind, some of these features include:
- In-Launcher username changing
- In-Launcher skin manager, allowing multiple skins (up to 30) to be saved and easily changed.
- Launcher Discord Rich Presence, showing your current state in the launcher on your Discord profile (needs the Discord desktop client to be active).
